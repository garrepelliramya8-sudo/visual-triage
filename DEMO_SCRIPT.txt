# ─────────────────────────────────────────────────────
# DEMO 1: Full Happy Path (GREEN severity)
# ─────────────────────────────────────────────────────

# Step 1: Send a routine symptom description
USER: I have a dry, slightly itchy rash on my forearm that appeared about a week ago. No pain or fever.

# Expected: security_checkpoint passes (no PII, no injection)
# Expected: symptom_analyzer classifies as GREEN / dermatology
# Expected: hitl_consent_gate asks for HIPAA consent + ZIP code

# Step 2: Provide consent and ZIP code
USER: I consent for ZIP 94043

# Expected: hitl_consent_gate approves, routes to provider_locator
# Expected: provider_locator queries MCP tools for dermatology in 94043
# Expected: final_output displays Silicon Valley Dermatology Center with wait times and reviews
